Agent CRM — a marketing landing page for an agentic CRM product, built with Next.js 16, Tailwind CSS v4, and Framer Motion.

Inspired by [attio.com](https://attio.com) design language. Extracted as a skill-building exercise in high-end SaaS marketing page construction.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Static export to Cloudflare Pages via `output: 'export'`.

```bash
npm run build   # outputs to out/
```

## Stack

- **Next.js 16** (static export)
- **React 19** + Framer Motion
- **Tailwind CSS v4** (`@tailwindcss/postcss`)
- **TypeScript**

## Sections

Hero → Logo cloud → Platform tabs → Live demo → Universal Context → Customer stories → Changelog → CTA → Footer
